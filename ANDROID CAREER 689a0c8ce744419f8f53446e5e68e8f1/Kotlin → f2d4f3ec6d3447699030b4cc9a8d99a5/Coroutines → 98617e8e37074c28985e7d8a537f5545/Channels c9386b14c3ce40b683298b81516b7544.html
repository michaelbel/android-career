Channels

https://kotlinlang.org/docs/channels.html

Channel

Каналы позволяют передавать потоки данных от одной корутины к другой. Похожи на BlockingQueue но без блокирующих операций put и take.

suspend fun main() = coroutineScope {
    val channel = Channel<String>()
    launch {
        val users = listOf("Tom", "Bob", "Sam")
        for (user in users) {
            println("Sending $user")
            channel.send(user)
        }
    }
    repeat(3) {
        val user = channel.receive()
        println("Received: $user")
    }
    println("End")
}

// Sending Tom
// Received: Tom
// Sending Bob
// Received: Bob
// Sending Sam
// Received: Sam
// End

send

Отправить объект в канал. Можно вызвать только из корутины.

receive

Получить данные из канала.

receiveAsFlow

Получить данные из канала и преобразовать их во Flow.

close(Throwable)

Закрыть канал чтобы больше не получать данные.

producer

Представляет билдер корутины который создает корутину в которой передаются данные в канал.

fun CoroutineScope.getUsers(): ReceiveChannel<String> = produce {
    val users = listOf("Tom", "Bob", "Sam")
    for (user in users) {
        send(user)
    }
}

consumeEach

Нужен для получения данных из канала. Заменяет цикл for.

val users = getUsers()
users.consumeEach { user -> println(user) }
