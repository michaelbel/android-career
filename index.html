Привет! Это роадмап «Андоид Карьера» для подготовки к собесу на Senior Android Developer.

На текущий момент включает 9 разделов. Каждый раздел - отдельная Notion-страница.

• стрелка → в названии страницы указывает на наличие вложенных страниц.

• страницы содержат заголовки в 2 стилях: Просто H3 для описания чего-то общего (например: Annotations) и H3 + Monospace для компонентов из SDK (например: CharSequence).
